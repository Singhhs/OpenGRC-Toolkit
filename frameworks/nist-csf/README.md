## CSF 2.0 Overview

### Functions

| Function | Purpose |
|----------|---------|
| **Govern (GV)** | *New in 2.0* - Establish and monitor cybersecurity risk management strategy, expectations, and policy |
| **Identify (ID)** | Understand organizational context and cybersecurity risks |
| **Protect (PR)** | Implement appropriate safeguards |
| **Detect (DE)** | Identify cybersecurity events |
| **Respond (RS)** | Take action regarding detected events |
| **Recover (RC)** | Maintain resilience and restore capabilities |
