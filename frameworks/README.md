## Framework Selection Guide

### When to Use ISO 27001
- International clients or operations
- Need for formal certification
- Comprehensive ISMS implementation
- B2B relationships requiring certification proof

### When to Use NIST CSF
- US-based organizations
- Critical infrastructure sectors
- Want flexible, risk-based approach
- No formal certification needed

### When to Use SOC 2
- SaaS or service providers
- Customer-facing audit requirements
- US market focus
- Need attestation report for clients

### When to Use GDPR
- Processing EU resident data
- Operating in EU markets
- Handling personal data
- Need data protection compliance
