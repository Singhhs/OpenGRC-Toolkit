# ISO 27001:2022 Annex A Controls

Reference guide for implementing ISO 27001:2022 Annex A controls.

## Control Structure

ISO 27001:2022 contains **93 controls** organized into **4 themes**:

| Theme | Number | Controls |
|-------|--------|----------|
| Organizational Controls | 5 | 37 controls |
| People Controls | 6 | 8 controls |
| Physical Controls | 7 | 14 controls |
| Technological Controls | 8 | 34 controls |
