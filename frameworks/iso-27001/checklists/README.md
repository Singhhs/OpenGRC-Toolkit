## How to Use

### For Implementation
1. Start with the [Implementation Checklist](../../../checklists/iso27001-implementation-checklist.md)
2. Work through each clause systematically
3. Use the [Security Baseline Checklist](../../../checklists/security-baseline-checklist.md) for technical controls

### For Internal Audit
1. Use the [Audit Checklist](../../../checklists/iso27001-audit-checklist.md) as your guide
2. Sample evidence for each control area
3. Document findings and nonconformities

### For Certification Preparation
1. Complete all implementation items
2. Run a full internal audit
3. Address all nonconformities
4. Verify documentation completeness
