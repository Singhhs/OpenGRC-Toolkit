### Clauses (Management System Requirements)

| Clause | Title | Key Documents |
|--------|-------|---------------|
| 4 | Context of the Organization | [ISMS Scope](./isms-scope.md), Stakeholder Analysis |

| Theme | Controls | Description | Policies |
|-------|----------|-------------|----------|
| Organizational (A.5) | 37 controls | Policies, roles, responsibilities | [View all policies](../../policies/README.md) |

See [Controls Reference](./controls/README.md) for detailed control-by-control guidance.

## Implementation Roadmap

### Phase 1: Foundation (Weeks 1-4)
- [ ] Define [ISMS scope](./isms-scope.md)
- [ ] Establish [security policy](../../policies/information-security-policy.md)
- [ ] Assign roles and responsibilities
- [ ] Conduct initial [risk assessment](../../risk-assessment/README.md)

### Phase 2: Risk Treatment (Weeks 5-8)
- [ ] Complete [risk treatment plan](./risk-treatment-plan.md)
- [ ] Develop [Statement of Applicability](./statement-of-applicability.md)
- [ ] Implement priority controls
- [ ] Create core procedures

### Phase 3: Implementation (Weeks 9-16)
- [ ] Implement remaining controls
- [ ] Develop all required documentation
- [ ] Conduct security awareness training
- [ ] Establish monitoring processes

### Phase 4: Verification (Weeks 17-20)
- [ ] Conduct [internal audit](./procedures/internal-audit-program.md)
- [ ] Perform [management review](../../templates/management-review-agenda.md)
- [ ] Address nonconformities
- [ ] Prepare for certification audit

## Certification Process

```
Stage 1 Audit          Stage 2 Audit         Surveillance
     │                      │                     │
     ▼                      ▼                     ▼
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│ Documentation│  →   │ Implementation│  →   │   Annual    │
│   Review    │       │   Audit     │       │   Audits    │
└─────────────┘       └─────────────┘       └─────────────┘
```

### Stage 1 (Documentation Review)
- Review ISMS documentation
- Verify scope and policies
- Check mandatory documents
- Identify any gaps

### Stage 2 (Implementation Audit)
- Verify implementation of controls
- Interview personnel
- Review evidence and records
- Identify nonconformities

### Surveillance Audits
- Annual audits to maintain certification
- Sample of controls reviewed each year
- Full recertification every 3 years
