## Rating Scale

| Rating | Description | Criteria |
|--------|-------------|----------|
| **0 - Not Implemented** | Control does not exist | No policy, process, or technology |
| **1 - Initial** | Control is ad-hoc | Informal, undocumented processes |
| **2 - Developing** | Control partially implemented | Documented but inconsistent execution |
| **3 - Defined** | Control fully implemented | Documented, consistent, some evidence |
| **4 - Managed** | Control monitored and measured | Regular reviews, metrics tracked |
| **5 - Optimized** | Control continuously improved | Automated, benchmarked, audit-ready |

**Target for SOC 2:** Rating of 4 or higher for all in-scope criteria
