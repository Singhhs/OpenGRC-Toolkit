## Template Conventions

All templates use:
- `[BRACKETED TEXT]` for placeholders to replace
- `<!-- CUSTOMIZE: notes -->` for guidance comments
- Consistent formatting and structure
- Markdown format for easy editing

### Common Placeholders

| Placeholder | Replace With |
|-------------|--------------|
| `[ORGANIZATION NAME]` | Your company name |
| `[DATE]` | Effective or revision date |
| `[VERSION]` | Document version number |
| `[OWNER]` | Document owner role/title |
| `[DEPARTMENT]` | Relevant department name |
| `[ROLE]` | Specific job title |
| `[NAME]` | Person's name |
| `[PHONE]` | Phone number |
| `[EMAIL]` | Email address |

---

## Usage Tips

1. **Download/copy** the template you need
2. **Search and replace** all placeholders
3. **Review** customization notes and adjust content
4. **Remove** sections that don't apply
5. **Add** organization-specific content
6. **Review** with stakeholders before finalizing
