# Crisis Communication Templates

Ready-to-use templates for incident and crisis communication.

## Available Templates

### Internal Communication

| Template | Purpose |
|----------|---------|
| [Internal Alert](./internal-alert-template.md) | Initial incident notification to teams |
| [Executive Briefing](./executive-briefing-template.md) | Status update for leadership |
| [Employee Communication](./employee-communication-template.md) | All-staff notification |
| [Incident Report](./incident-report-template.md) | Final incident documentation |

### External Communication

| Template | Purpose |
|----------|---------|
| [Customer Notification](./customer-notification-template.md) | Notify affected customers |
| [Regulatory Notification](./regulatory-notification-template.md) | Report to authorities (GDPR, etc.) |
| [Press Statement](./press-statement-template.md) | Media response |
| [Partner Notification](./partner-notification-template.md) | Alert business partners |

### Documentation

| Template | Purpose |
|----------|---------|
| [Incident Intake Form](./incident-intake-form.md) | Initial incident capture |
| [Lessons Learned](./lessons-learned-template.md) | Post-incident review |

## Usage Guidelines

1. Select the appropriate template
2. Replace all placeholder text `[IN BRACKETS]`
3. Remove sections that don't apply
4. Review with legal/communications before sending externally
5. Keep copies of all communications sent
